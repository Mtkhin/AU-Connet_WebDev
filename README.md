## About our project
AU-Connet is a web application that allows users to connect, manage events, and interact through a modern web platform.
The system includes a frontend interface and a backend API that handles data, authentication, and application logic.

## Our Features
User registration and login
Create and manage clubs 
Join and leave clubs 
Create and manage events under clubs 
View upcoming events 
Manage club memberships
Basic Role Sys

## Tech Stacks
Frontend: Next.js 
Backend: Next.js API Routes ( REST API ) 
Database: MongoDB
